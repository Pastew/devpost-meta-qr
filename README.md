# Devpost Meta Horizon QR

A Chrome extension that automatically generates and displays a QR code for "Try it out" links (specifically Meta Horizon World links) on Devpost project pages. This makes it easy to open the links directly on your mobile device.

## Installation

1.  **Download via this specific link**:
    [**Download devpost-meta-qr.zip**](https://github.com/Pastew/devpost-meta-qr/releases/download/v1/devpost-meta-qr.zip)

2.  **Unzip** the downloaded file to a folder on your computer.

3.  Open Google Chrome and navigate to `chrome://extensions`.

4.  Enable **Developer mode** by toggling the switch in the top-right corner.

5.  Click the **Load unpacked** button that appears.

6.  Select the folder where you unzipped the extension files.

7.  Go to a Devpost page with a Meta Horizon link (e.g., [Donâ€™t Wake My Pet: Draw the Symbols](https://devpost.com/software/signs-tqbvfs)) to see the QR code!
